Default p2constants values:
player_speed = 3
movement_distance = 20
slow_turn_angle = 15

MAKE SURE TO PRESS "X" TO EXIT PROGRAM OR ELSE IT WILL NOT START AGAIN UNLESS YOU MANUALLY EXIT IN TERMINAL "exit()"
Note: The turtle/player has tank movement, meaning that "A" and "D" will rotate the turtle/player 90 degrees, not move horizontally.
 W = Move player forward by "movement_distance" value.  (Can be held down)
 S = Move player backward by "movement_distance" value. (Can be held down)
 A = Rotate player 90 degrees counterclockwise.         (Can be held down)
 D = Rotate player 90 degrees clockwise.                (Can be held down)
 Z = Undo last action.                                  (Can be held down)
 C = Open colour picker.                                (Cannot be held down)
 V = Clears all player drawings.                        (Cannot be held down)
 X = Exits program safely.                              (Cannot be held down)
 P = Takes a screenshot. (Work In Progress)             (DO NOT HOLD DOWN)

FOR VS CODE USERS:

- Then you may run the "twesbozotesting.py" file
